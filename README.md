# 掌上仓库 APP

> 面向工厂仓库的移动端仓储管理系统，支持扫码入库出库、库存盘点、订单管理

基于 Expo 54 + React Native + Express.js 构建的跨平台移动应用。

---

## 产品简介

**掌上仓库**是一款专为仓库管理人员设计的移动端工具，主要解决以下场景：

- 📥 **扫码入库** - 扫描物料二维码，自动解析型号、批次、数量等信息
- 📤 **扫码出库** - 扫描出库，支持订单匹配和批次管理  
- 🔍 **库存盘点** - 定期盘点，支持扫码快速核对库存差异
- 📋 **订单管理** - 订单列表、时间筛选、仓库筛选、数据统计
- 🏭 **仓库管理** - 多仓库切换，数据隔离
- 📊 **数据导出** - 一键导出 Excel，与 PC 端无缝衔接

---

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端框架 | Expo 54 + React Native | 跨平台移动应用 |
| 开发语言 | TypeScript | 类型安全 |
| 后端服务 | Express.js | API 接口服务 |
| 本地存储 | AsyncStorage + SQLite | 数据持久化 |
| 项目架构 | pnpm Monorepo | 前后端统一管理 |

## 核心功能

### 扫码入库
- 支持物料二维码扫描
- 自动解析型号、批次、数量等信息
- 扫码队列机制，连续扫码自动处理
- 三重查重机制，避免重复入库

### 扫码出库
- 扫描出库，支持订单匹配
- 订单号、仓库、扫码记录持久化保存
- App 重启后自动恢复扫码状态

### 库存盘点
- 扫码盘点，核对实际库存
- 自动记录盘点差异
- 盘点数据按仓库隔离

### 订单管理
- 订单列表展示
- 时间筛选（今天/近三天/近七天/全部）
- 仓库筛选
- 统计面板（订单数量、物料数量）
- 物料汇总和详情查看

### 仓库管理
- 多仓库支持
- 仓库切换，数据隔离
- 默认仓库设置

### 规则配置
- 自定义扫码解析规则
- 支持多种分隔符识别

## 快速开始

### 环境要求
- Node.js 18+
- pnpm 8+

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
# 启动前端（端口 5000）
cd client && npx expo start

# 启动后端（端口 9091）
cd server && pnpm dev
```

### 构建 APK
```bash
pnpm build
```
